# PenPal Draw - Agent Notes

## Project Overview

A cross-platform drawing app for the XPPen Artist 15.6 Pro V2 Pen Display.
Built with Tauri (Rust + Web technologies).

## Architecture

- **Frontend**: HTML5 Canvas with Pointer Events API for pen input
- **Backend**: Rust with Tauri for native window management
- **Build System**: Cargo (Rust) + npm (Node.js)

## Key Features Implemented

1. **Pressure Sensitivity**
   - Uses Pointer Events API (`pointerdown`, `pointermove`, `pointerup`)
   - Reads `e.pressure` from pen events (0.0 - 1.0)
   - Brush size scales with pressure when enabled

2. **Multi-Monitor Support**
   - Rust backend queries available monitors
   - Frontend dropdown to select target display
   - Can move window to specific monitor coordinates
   - Fullscreen mode support

3. **Tool System**
   - `ToolManager` class handles all tools and brushes
   - 7 preset brushes: Round, Square, Soft, Pencil, Marker, Spray, Charcoal
   - 5 tools: Eraser, Smudge/Blend, Blur, Dodge, Burn
   - 10 blend modes for brushes
   - Custom brush creator with preview
   - Tool-specific UI (shows relevant controls per tool)

4. **Advanced Tools**
   - **Eraser**: Pixel, Brush, Block modes
   - **Smudge**: Blends existing paint with adjustable strength
   - **Blur**: Box blur effect
   - **Dodge/Burn**: Lighten/darken like photography tools
   - **Blend Modes**: Normal, Multiply, Screen, Overlay, Soft Light, Hard Light, Color Dodge, Color Burn, Difference, Exclusion

5. **Canvas Management**
   - Dynamic resizing with content preservation
   - Undo/Redo system with stroke history
   - Save as PNG

6. **Shortcut Keys (XPPen Tablet)**
   - `ShortcutManager` class handles key bindings
   - Modal UI for configuration (вЊЁпёЏ Keys button)
   - Auto-detect mode for quick setup
   - Per-key action assignment
   - Supports combo keys (Ctrl+F1, etc.)
   - Actions: brush types, sizes, colors, undo/redo, save, eraser, fullscreen
   - Settings persist to localStorage

7. **In-App Help System**
   - `HelpSystem` class manages help modal
   - Sidebar navigation with 8 help sections
   - Search functionality with text highlighting
   - F1 keyboard shortcut to open
   - Comprehensive documentation for all features
   - Tips & Tricks section
   - Troubleshooting guides

## Building

### Prerequisites
- Node.js 18+
- Rust (via rustup)
- Windows: Visual Studio Build Tools

### Development
```bash
npm install
npm run dev
```

### Production Build
```bash
npm run build
```

Or use `setup.bat` on Windows for one-click build.

## XPPen Specific Notes

- App detects pen input via Pointer Events `pointerType === 'pen'`
- Pressure is read from `PointerEvent.pressure`
- Requires XPPen drivers with "Windows Ink" enabled
- Second monitor (XPPen) is auto-selected in dropdown if not primary

## Known Limitations

- Undo system stores stroke data in memory (may use RAM for long sessions)
- No layer support (single canvas)
- No brush textures (solid circles only)
- ICNS icon for macOS needs manual conversion
