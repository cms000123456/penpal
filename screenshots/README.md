## Screenshot Guidelines

- **Resolution:** 1920x1080 or higher
- **Format:** PNG (for transparency support)
- **Compression:** Use reasonable compression (target < 500KB per image)
- **Content:** Show realistic usage, not just empty canvas
- **Annotations:** Consider adding numbered callouts for key features

## How to Take Screenshots

### Windows
1. Press `Win + Shift + S` for snipping tool
2. Or use `Win + PrintScreen` to save to Pictures folder

### macOS
1. Press `Cmd + Shift + 4` for selection tool
2. Or `Cmd + Shift + 5` for more options

### Linux
1. Use `gnome-screenshot` or `flameshot`
2. Or `PrintScreen` key

## Adding Screenshots

1. Take screenshot
