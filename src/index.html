🚀 Quick Start

Welcome to PenPal Draw - a drawing app designed for your XPPen Artist 15.6 Pro V2!

Getting Started (3 Steps)

  1. Connect your tablet - Make sure your XPPen is connected and drivers are installed
  2. Select your display - Use the "Display" dropdown to move the app to your pen display
  3. Start drawing! - Pick a brush and draw on the canvas

Basic Controls

  • Left click + drag - Draw with current tool
  • Pressure - Press harder for thicker/opaque strokes (with compatible pens)
  • Mouse wheel - Adjust brush size (when implemented)
  • Ctrl + Z - Undo
  • Ctrl + Y - Redo
  • Ctrl + S - Save image
💡 Tip: Click the Fullscreen button for the best drawing experience on your XPPen!

🖌️ Tools & Brushes

Brushes

Round - Standard circular brush, good for general drawing
Square - Hard-edged square, good for geometric shapes
Soft - Soft-edged circular, good for shading
Pencil - Textured stroke, simulates real pencil
Marker - Broad, marker-like strokes
Spray - Airbrush effect with particles
Charcoal - Grainy texture for sketching

Tools

Eraser - Remove paint. Modes: Pixel (hard), Brush (soft), Block (square)
Smudge/Blend - Blend existing colors together. Adjust strength with slider
Blur - Soften areas of your image
Dodge - Lighten areas (like photography dodge tool)
Burn - Darken areas (like photography burn tool)

Brush Settings

  • Size - Overall brush diameter (1-100)
  • Opacity - Transparency of strokes (1-100%)
  • Pressure - Enable/disable pressure sensitivity
  • Hardness - Edge softness (0-100%)
  • Spacing - Distance between brush stamps

📑 Layers

PenPal Draw now supports layers! Layers let you draw on separate transparent sheets that can be stacked, hidden, merged, and reordered.

Layer Panel

The layer panel appears on the right side of the screen. Each layer shows a thumbnail preview of its contents.

Layer Controls

  • Add (+) - Create a new empty layer
  • Delete (🗑️) - Remove the selected layer
  • Merge (⮃) - Combine selected layer with the one below
  • Opacity slider - Adjust layer transparency
  • Visibility (eye) - Show/hide layer

Working with Layers

  • Click a layer to select it - new strokes go on the active layer
  • Drag layers to reorder them
  • Double-click layer name to rename
  • Background layer is always at the bottom and filled with white
💡 Tip: Use layers to separate sketch, lineart, and coloring for non-destructive editing!

🎨 XPPen Setup

Required Setup

  1. Install XPPen drivers from xp-pen.com/download
  2. Connect your Artist 15.6 Pro V2 via USB and HDMI
  3. Open XPPen Tablet software
  4. Enable Windows Ink in pen settings

Configure Shortcut Keys

  1. Open XPPen Tablet software
  2. Go to Shortcut Keys tab
  3. Set each key to send keyboard shortcuts (e.g., Ctrl+F1 for K1)
  4. In PenPal Draw, click ⌨️ Keys and map the shortcuts

Troubleshooting Pressure

  • Ensure "Pressure" checkbox is enabled in toolbar
  • Check that Windows Ink is enabled in XPPen drivers
  • Try restarting the app after driver changes
  • Test with the pressure indicator in the toolbar

✨ Custom Brushes

Creating a Custom Brush

  1. Click the + button next to the tool selector
  2. Adjust the parameters:
  • Type - Base shape (round, square, soft, textured, pattern)
  • Hardness - Edge softness (0% = very soft, 100% = hard edge)
  • Spacing - Distance between brush stamps (lower = smoother)
  • Angle - Rotation of brush tip
  • Roundness - Compress circle to ellipse
  • Min Size - Size when pressure is minimum
  • Texture - Add noise, grain, dots, or lines
  1. Enter a name for your brush
  2. Click Save Brush
💡 Tip: Custom brushes are saved automatically and persist between sessions!

🔧 Troubleshooting

Pressure Not Working

  • Check "Pressure" checkbox in toolbar
  • Ensure Windows Ink is enabled in XPPen drivers
  • Restart the app
  • Test with pressure indicator (should show colors)

App Won't Move to Pen Display

  • Ensure XPPen is recognized as a monitor in Windows
  • Manually drag window to pen display
  • Use Fullscreen button once positioned

Lag or Stuttering

  • Reduce brush size
  • Close other applications
  • Check CPU/GPU usage
  • Try simpler brushes (Round instead of Charcoal)

Smudge/Blur Not Working

  • These tools only work on existing paint
  • Draw something first, then use smudge/blur
  • Try increasing the strength slider

💡 Tips & Tricks

Quick Color Pick: Use the color picker tool (shortcut key) to sample colors from your canvas
Pressure Practice: The pressure indicator in the toolbar helps you learn pen control - yellow (light), green (medium), red (heavy)
Smooth Lines: Lower the spacing value in custom brushes for smoother strokes
Texture Brushes: Combine textures with low hardness for realistic pencil/charcoal effects
Undo History: The undo system stores complete stroke data - you can undo/redo freely
Save Versions: Use Ctrl+S frequently - each save has a timestamp in the filename
Fullscreen Drawing: Use Fullscreen mode for the most immersive experience
Web Testing: Run run-web.bat to test in browser before building desktop app
